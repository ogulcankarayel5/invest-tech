## Dependencies

- [Vite]
- [React]
- [TypeScript] - more strict
- [React Router]
- [Redux Toolkit]
- [ESLint]
- [Prettier]
- [PrimeReact]
- [ReactLoading]
- [axios]

## Installation

```bash
git clone https://github.com/ogulcankarayel5/invest-tech.git
cd invest-tech
create .env file on the root directory and add VITE_API_KEY value to call the api
yarn install
yarn dev
```
